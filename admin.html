Alur Kerja

Workflow update menu harian

  1. Buka admin.html.
  2. Tambah, edit, hapus, atau ubah status menu.
  3. Atur menu ke hari ini atau besok.
  4. Klik tombol unduh JSON.
  5. Ganti file data/menu.json di repository GitHub.
